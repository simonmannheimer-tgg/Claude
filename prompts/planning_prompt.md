# Planning Prompt

Use this prompt to decompose a task before starting work.

---

**Goal:** [What needs to be accomplished?]

**Constraints:** [What must not be violated? Time, resources, dependencies, scope limits.]

**Steps:** [Ordered list of actions to take.]
1.
2.
3.

**Dependencies:** [Which steps depend on which?]

**Risks:** [What could go wrong? How would you handle it?]

**Success criteria:** [How will you know the task is done correctly?]
